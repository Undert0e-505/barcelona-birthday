⚙️ How It Works

Built as a single-page app with no framework. Votes stored in KVdb.io (free, CORS-friendly key-value store). Each person gets a unique URL with an unguessable code. Results show all three voters side-by-side. Deployed on GitHub Pages — push to main = live.
