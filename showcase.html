🇪🇸 Barcelona Birthday

A surprise trip planner with live voting

👀 Showcase — Voting Disabled

This is a demo view. Each voter gets a personal link to cast their votes. Voting buttons are disabled here.

⚙️ How It Works

Built as a single-page app with no framework. Votes are stored in KVdb.io (free, CORS-friendly key-value store). Each person gets a unique URL with an unguessable code. Results show all three voters side-by-side. Deployed on GitHub Pages — push to main = live.
